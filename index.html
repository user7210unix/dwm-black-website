White DWM Dotfiles 👀

A sleek and minimal setup with all dotfiles for your DWM environment.

Features

  • Window Manager 🍱 DWM
  • Application Launcher 🚀 Rofi
  • Shell 🐚 Fish
  • Compositor 🍧 Picom

🚀 Resource Efficiency

Optimized for performance and minimal resource usage. This lightweight configuration maximizes system resources for essential tasks and keeps everything running smoothly.

🔑 Key Bindings

  • $mod+q – KILL
  • $mod+Return – ST
  • $mod+f – FIREFOX
  • $mod+d – DMENU
  • $mod+e – PCMANFM
